About Readable Spokable PDF

What is this?

Readable Spokable PDF is a browser-based tool that transforms technical PDFs into natural, spoken-friendly documents optimized for text-to-speech (TTS) consumption. It uses Google's Gemini AI to intelligently convert complex technical content into easy-to-listen narratives.

Why did we build this?

Technical documentation, research papers, and educational materials are often filled with code blocks, mathematical notation, tables, and figures that don't translate well to text-to-speech systems. When you try to listen to these documents using TTS apps, you end up hearing code read line-by-line, tables read cell-by-cell, and math symbols spoken as raw characters.

This tool solves that problem by:

  • Converting code blocks into descriptive explanations
  • Transforming tables into narrative sentences
  • Converting mathematical notation into spoken form
  • Describing figures and images clearly
  • Adding natural transitions and rhythm for better listening

How does it work?

  1. Text Extraction: We extract text from your PDF using PDF.js
  2. Intelligent Chunking: The text is split into manageable batches with overlap
  3. AI Transformation: Each batch is sent to Google's Gemini AI with specialized prompts
  4. Smart Assembly: Transformed batches are reassembled into a coherent document
  5. PDF Generation: A new, TTS-friendly PDF is created with your preferred formatting

Privacy & Security

Your privacy is important to us:

  • All processing happens in your browser - no server uploads
  • Your API key is stored only in your browser's LocalStorage
  • Files are stored only in your browser's IndexedDB
  • No data is sent to any third party except Google AI Studio
  • You can erase all data at any time from Settings

Technology Stack

  • PDF.js: Mozilla's PDF rendering library for text extraction
  • jsPDF: PDF generation library
  • Google Gemini AI: Advanced language model for text transformation
  • IndexedDB: Browser storage for persistence
  • Vanilla JavaScript: No frameworks, just modern ES6+

Open Source
