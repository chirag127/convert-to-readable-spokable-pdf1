Privacy Policy

Last updated: November 19, 2025

Overview

Readable Spokable PDF is a client-side web application that runs entirely in your browser. We are committed to protecting your privacy and being transparent about how your data is handled.

Data Collection

We do NOT collect, store, or transmit any of your data to our servers because we don't have any servers. All processing happens locally in your browser.

Local Storage

The application stores data locally in your browser using:

  • LocalStorage: API keys and UI preferences (dark mode, welcome banner state)
  • IndexedDB: Uploaded PDF files, processed text, batches, logs, and settings

This data never leaves your device unless you explicitly export it.

Third-Party Services

The application communicates with the following third-party services:

Google AI Studio (Gemini API)

  • Purpose: Text transformation using AI
  • Data sent: Text extracted from your PDFs, your API key
  • Privacy policy: Google Privacy Policy
  • Your API key is sent directly from your browser to Google's servers
  • We do not intercept, log, or store your API communications

CDN Services

  • Purpose: Loading PDF.js and jsPDF libraries
  • Provider: Cloudflare CDN
  • Data sent: Standard HTTP request headers

API Keys

Your Google AI Studio API key is stored only in your browser's LocalStorage. It is never transmitted to any server other than Google AI Studio endpoints. You can clear your API key at any time from the Settings panel.

File Processing

When you upload a PDF:

  • The file is read and processed entirely in your browser
  • Text is extracted using PDF.js (client-side)
  • Extracted text is sent to Google AI Studio for transformation
  • The original PDF and transformed text are stored in your browser's IndexedDB
  • No files are uploaded to our servers (we don't have any)

Logs

The application maintains local logs of API requests and processing events for debugging purposes. These logs are stored in IndexedDB and never leave your browser. API keys in logs are automatically redacted. You can view, export, or clear logs from the Diagnostics page.

Cookies

This application does not use cookies.

Data Deletion

You can delete your data at any time:

  • Individual files: Use the delete button on stored files
  • API keys: Use "Clear API Keys" in Settings
  • All data: Use "Erase All Data" in Settings â†’ Advanced
  • Browser data: Clear your browser's LocalStorage and IndexedDB manually

Security

While we implement security best practices, please note:

  • Your API key is stored in plain text in LocalStorage (browser limitation)
  • Anyone with access to your device can potentially access stored data
  • Use device-level security (passwords, encryption) to protect your data
  • Don't share your API key with others
  • Regularly clear stored files if they contain sensitive information

Children's Privacy

This service is not directed to children under 13. We do not knowingly collect information from children.

Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page with an updated revision date.

Contact

If you have questions about this privacy policy, please contact us through the Contact page.
